# مشروع التعرف على الألوان باستخدام OpenCV

## وصف المشروع

هذا المشروع عبارة عن نظام للتعرف على الألوان في الوقت الحقيقي باستخدام مكتبة **OpenCV** ولغة **Python**.

يقوم البرنامج بفتح كاميرا الجهاز وتحليل الصورة المباشرة، ثم يكتشف الألوان التالية:

- 🟢 الأخضر (Green)
- 🔴 الأحمر (Red)
- 🔵 الأزرق (Blue)
- 🟡 الأصفر (Yellow)

عند اكتشاف أي جسم يحمل أحد هذه الألوان، يقوم البرنامج برسم مستطيل حوله مع عرض اسم اللون على الشاشة.

---

# البرامج المستخدمة

- Python 3.11
- OpenCV
- NumPy
- Anaconda
- Visual Studio Code

---

# متطلبات التشغيل

قبل تشغيل المشروع تأكد من تثبيت Python من خلال Anaconda، ثم ثبّت المكتبات المطلوبة بالأمر التالي:

```bash
pip install -r requirements.txt
```

---

# طريقة تشغيل المشروع

بعد تثبيت المتطلبات، شغّل البرنامج بالأمر التالي:

```bash
python color_recognition.py
```

ستفتح كاميرا الجهاز مباشرة، وسيبدأ البرنامج بالتعرف على الألوان في الوقت الحقيقي.

للخروج من البرنامج اضغط على المفتاح:

```text
Q
```

---

# آلية عمل المشروع

يعتمد المشروع على الخطوات التالية:

1. تشغيل كاميرا الجهاز.
2. التقاط الفيديو بشكل مستمر.
3. تحويل الصورة من نظام الألوان **BGR** إلى **HSV**.
4. تحديد نطاق كل لون باستخدام قيم HSV.
5. إنشاء قناع (Mask) لكل لون.
6. استخراج حدود الأجسام (Contours).
7. رسم مستطيل حول الجسم المكتشف.
8. عرض اسم اللون على الشاشة.
